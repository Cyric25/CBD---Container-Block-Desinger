Test-Anweisungen

  1. Dropdown-Menü: Klicken Sie auf das Zahnrad-Symbol in der oberen rechten Ecke jedes Containers
  2. Collapse: Klicken Sie auf "Einklappen/Ausklappen" im Dropdown-Menü
  3. Text kopieren: Klicken Sie auf "Text kopieren" - es sollte eine Erfolgsmeldung erscheinen
  4. Screenshot: Klicken Sie auf "Screenshot" - es sollte ein Download gestartet werden
  5. HTML-Elemente: Alle Buttons, Eingabefelder und andere HTML-Elemente sollten funktionieren
